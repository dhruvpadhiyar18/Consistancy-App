1. Overview

Consistancy is a privacy-first educational application designed to help users build daily discipline while following a structured 180-day AI/ML and Software Engineering roadmap.

We respect your privacy deeply. This app does not collect, store, transmit, or share any personal data.

2. Data Collection

Consistancy does NOT collect:

  • Name, email address, phone number
  • Login credentials or account data
  • Location data
  • Device identifiers
  • Usage analytics or tracking data

3. Data Storage

All user progress (tasks, streaks, quiz results, milestones) is stored locally on the user's device using Android SharedPreferences.

  • No cloud storage
  • No backend servers
  • No external databases

If the app is uninstalled or device data is cleared, all progress is permanently lost.

4. Permissions Used

The app requests the following permissions:

  • Internet: To open third-party learning resources inside an in-app WebView
  • Notifications: To send daily learning reminders (Morning, Afternoon, Evening)
  • Vibration (Haptic): For tactile feedback on task completion
  • Boot Completed: To reschedule reminders after device restart

5. Third-Party Content

Consistancy may open third-party educational websites (such as YouTube, blogs, documentation) inside an in-app WebView.

We do not control, track, or monitor activity on these external websites.
